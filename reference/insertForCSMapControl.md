# Specify `ForCSMapControl` for Forest Carbon Succession (ForCS) extension

Specify `ForCSMapControl` for Forest Carbon Succession (ForCS) extension

## Usage

``` r
insertForCSMapControl(df)
```

## Arguments

- df:

  data.frame with one row and seven columns: `BiomassC`, `SDOMC`, `NBP`,
  `NEP`, `NPP`, `RH`, `ToFPS` toggles (each `0` or `1`).

## Value

data.frame
