# Specify the `SpeciesTable` block for the Biomass Browse extension

Specify the `SpeciesTable` block for the Biomass Browse extension

## Usage

``` r
insertBrowseSpeciesTable(df)
```

## Arguments

- df:

  `data.frame` with columns enumerated in `.browseSpeciesTableCols`.

## Value

Character string(s) to write to the suitable LANDIS-II input file.
