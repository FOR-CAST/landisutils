# Specify the outbreak-pattern block of a Climate BDA agent file

Specify the outbreak-pattern block of a Climate BDA agent file

## Usage

``` r
insertOutbreakPattern(agent)
```

## Arguments

- agent:

  `BDAAgent` object.

## Value

Character string(s) to write to the suitable LANDIS-II input file.
