# Specify Biomass Succession Extension `MinRelativeBiomass` Table

Specify Biomass Succession Extension `MinRelativeBiomass` Table

## Usage

``` r
insertMinRelativeBiomass(df = NULL)
```

## Arguments

- df:

  data.frame corresponding to `MinRelativeBiomass` table

## Value

Character string(s) to write to the suitable LANDIS-II input file.
