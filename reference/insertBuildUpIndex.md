# Specify Dynamic Fire Extension `BuildUpIndex`

Specify Dynamic Fire Extension `BuildUpIndex`

## Usage

``` r
insertBuildUpIndex(bui = FALSE)
```

## Arguments

- bui:

  Logical, indicating whether to turn on Build Up Index.

## Value

Character string(s) to write to the suitable LANDIS-II input file.
