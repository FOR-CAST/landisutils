# Specify the `DistanceVariables` block for the Landscape Habitat extension

Specify the `DistanceVariables` block for the Landscape Habitat
extension

## Usage

``` r
insertLandscapeHabitatDistanceVariables(df)
```

## Arguments

- df:

  `data.frame` with columns `Name`, `LocalVar`, `Transform`.

## Value

Character string(s) to write to the suitable LANDIS-II input file.
