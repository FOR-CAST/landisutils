# Prepare Dynamic Fire Extension `DynamicWeatherTable`

Prepare Dynamic Fire Extension `DynamicWeatherTable`

## Usage

``` r
prepDynamicWeatherTable(year, filename)
```

## Arguments

- year:

  Integer vector of simulation years.

- filename:

  Character vector of filenames corresponding to each of `year`.

## Value

data.frame
