# Specify the `HarvestImplementations` table

Specify the `HarvestImplementations` table

## Usage

``` r
insertHarvestImplementations(df)
```

## Arguments

- df:

  `data.frame` with required columns `MgmtArea`, `Prescription`,
  `HarvestArea` and optional columns `BeginTime`, `EndTime`.

## Value

Character string(s) to write to the suitable LANDIS-II input file.
