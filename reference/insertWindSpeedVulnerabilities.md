# Specify the `WindSpeedVulnerabilities` block for the Hurricane extension

Specify the `WindSpeedVulnerabilities` block for the Hurricane extension

## Usage

``` r
insertWindSpeedVulnerabilities(vulns)
```

## Arguments

- vulns:

  List of `WindSpeedVulnerability` objects (see
  [`windSpeedVulnerability()`](https://for-cast.github.io/landisutils/reference/windSpeedVulnerability.md)).

## Value

Character string(s) to write to the suitable LANDIS-II input file.
