# Construct an exit road type (where wood leaves the landscape)

Construct an exit road type (where wood leaves the landscape)

## Usage

``` r
exitRoadType(id, name)
```

## Arguments

- id:

  Integer. Road type ID.

- name:

  Character. Road type name (no whitespace).

## Value

A list of class `"ExitRoadType"`.
