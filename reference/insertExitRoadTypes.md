# Specify the `RoadTypesForExitingWood` table for the Forest Roads extension

Specify the `RoadTypesForExitingWood` table for the Forest Roads
extension

## Usage

``` r
insertExitRoadTypes(roads)
```

## Arguments

- roads:

  List of `ExitRoadType` objects.

## Value

Character string(s) to write to the suitable LANDIS-II input file.
