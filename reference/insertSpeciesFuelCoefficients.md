# Specify Dynamic Fuel Extension's Species Fuel Coefficients

Specify Dynamic Fuel Extension's Species Fuel Coefficients

## Usage

``` r
insertSpeciesFuelCoefficients(df)
```

## Arguments

- df:

  data.frame corresponding to Species Fuel Coefficients

## Value

Character string(s) to write to the suitable LANDIS-II input file.
