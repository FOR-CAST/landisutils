# Specify Dynamic Fire Extension `GroundSlopeFile`

Specify Dynamic Fire Extension `GroundSlopeFile`

## Usage

``` r
insertGroundSlopeFile(file)
```

## Arguments

- file:

  Character, specifying the path to the file.

## Value

Character string(s) to write to the suitable LANDIS-II input file.
