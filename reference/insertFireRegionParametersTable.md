# Specify Fire Region Parameters Table

Specify Fire Region Parameters Table

## Usage

``` r
insertFireRegionParametersTable(df)
```

## Arguments

- df:

  data.frame corresponding to Fire Region Parameters Table

## Value

Character string(s) to write to the suitable LANDIS-II input file.
