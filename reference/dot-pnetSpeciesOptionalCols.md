# Optional `PnETSpeciesParameters` columns

Optional columns of the `PnETSpeciesParameters` input file (§8). Many of
these parameters can alternatively be supplied as generic values in the
`PnETGenericParameters` file; species-level values always take
precedence.

## Usage

``` r
.pnetSpeciesOptionalCols
```

## Format

An object of class `character` of length 22.
