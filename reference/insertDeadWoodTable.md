# Specify `DeadWoodTable` for Social-Climate-Fire extension

Specify `DeadWoodTable` for Social-Climate-Fire extension

## Usage

``` r
insertDeadWoodTable(df = NULL)
```

## Arguments

- df:

  `data.frame` with columns `Species` and `Age`.

## Value

Character string(s) to write to the suitable LANDIS-II input file.
