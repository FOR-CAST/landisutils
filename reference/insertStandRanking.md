# Specify the `StandRanking` portion of a Biomass Harvest prescription

Specify the `StandRanking` portion of a Biomass Harvest prescription

## Usage

``` r
insertStandRanking(rx)
```

## Arguments

- rx:

  `HarvestPrescription` object.

## Value

Character string(s) to write to the suitable LANDIS-II input file.
