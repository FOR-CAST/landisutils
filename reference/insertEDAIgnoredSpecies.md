# Specify the optional `IgnoredSpecies` block for an EDA agent

Specify the optional `IgnoredSpecies` block for an EDA agent

## Usage

``` r
insertEDAIgnoredSpecies(species)
```

## Arguments

- species:

  (Optional) Character vector of species codes.

## Value

Character string(s) to write to the suitable LANDIS-II input file.
