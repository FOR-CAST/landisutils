# Specify `Ecoregions` and `EcoregionsMap` Files

Specify `Ecoregions` and `EcoregionsMap` Files

## Usage

``` r
insertEcoregionsFiles(files)
```

## Arguments

- files:

  character, specifying the paths to the ecoregions and ecoregions map
  files.

## Value

Character string(s) to write to the suitable LANDIS-II input file.
