# Allowed `PnETGenericParameters` keys

Names of generic PnET parameters that may appear in a
`PnETGenericParameters` input file (see §7 of the LANDIS-II
PnET-Succession v6.0 user guide). Any parameter listed here may instead
be set in the `PnETGenericDefaultParameters.txt` file shipped with the
extension; values supplied here override the defaults.

## Usage

``` r
.pnetGenericParameterKeys
```
