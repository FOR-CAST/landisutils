# Specify the `EDAInputFiles` block in the main EDA file

Specify the `EDAInputFiles` block in the main EDA file

## Usage

``` r
insertEDAInputFiles(files)
```

## Arguments

- files:

  Character vector of relative paths to per-agent input files.

## Value

Character string(s) to write to the suitable LANDIS-II input file.
