# Specify the `NeighborhoodVariables` block for the Landscape Habitat extension

Specify the `NeighborhoodVariables` block for the Landscape Habitat
extension

## Usage

``` r
insertLandscapeHabitatNeighborhoodVariables(df)
```

## Arguments

- df:

  `data.frame` with columns `Name`, `LocalVar`, `NeighborRadius`,
  `Transform`.

## Value

Character string(s) to write to the suitable LANDIS-II input file.
