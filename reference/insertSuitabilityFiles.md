# Specify the `SuitabilityFiles` block for habitat output extensions

Emits the `SuitabilityFiles` keyword followed by one path per line.

## Usage

``` r
insertSuitabilityFiles(files)
```

## Arguments

- files:

  List of `SuitabilityFile` objects.

## Value

Character string(s) to write to the suitable LANDIS-II input file.
