# Specify the optional `IgnoredSpecies` block

Specify the optional `IgnoredSpecies` block

## Usage

``` r
insertIgnoredSpecies(species)
```

## Arguments

- species:

  (Optional) Character vector of species codes.

## Value

Character string(s) to write to the suitable LANDIS-II input file.
