# Specify the `DerivedLocalVariables` block for the Landscape Habitat extension

Specify the `DerivedLocalVariables` block for the Landscape Habitat
extension

## Usage

``` r
insertLandscapeHabitatDerivedLocalVariables(x)
```

## Arguments

- x:

  Named character vector mapping derived-variable names to arithmetic
  expressions referencing local variables.

## Value

Character string(s) to write to the suitable LANDIS-II input file.
