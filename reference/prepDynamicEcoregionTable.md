# Prepare Dynamic Fire Extension `DynamicEcoregionTable`

Prepare Dynamic Fire Extension `DynamicEcoregionTable`

## Usage

``` r
prepDynamicEcoregionTable(year = NULL, filename = NULL)
```

## Arguments

- year:

  Integer vector of simulation years

- filename:

  Character vector of filenames corresponding to each of `year`.

## Value

data.frame
