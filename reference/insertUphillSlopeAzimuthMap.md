# Specify Dynamic Fire Extension `UphillSlopeAzimuthMap`

Specify Dynamic Fire Extension `UphillSlopeAzimuthMap`

## Usage

``` r
insertUphillSlopeAzimuthMap(file)
```

## Arguments

- file:

  Character, specifying the path to the file.

## Value

Character string(s) to write to the suitable LANDIS-II input file.
