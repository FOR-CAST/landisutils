# Specify the `ClimateVariables` block for the Landscape Habitat extension

Specify the `ClimateVariables` block for the Landscape Habitat extension

## Usage

``` r
insertLandscapeHabitatClimateVariables(df)
```

## Arguments

- df:

  `data.frame` with columns `Name`, `Year`, `Months`, `Source`,
  `ClimateVar`, `Transform`.

## Value

Character string(s) to write to the suitable LANDIS-II input file.
