# Default Hurricane wind-speed mortality curve

Matches the Trevor / Sitka example used in the upstream
`Core8-Hurricane3.0` test input.

## Usage

``` r
defaultHurricaneMortalityCurve()
```

## Value

Named numeric vector mapping wind-speed thresholds to mortality
probabilities.
