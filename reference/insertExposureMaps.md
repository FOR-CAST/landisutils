# Specify the `ExposureMaps` block for the Hurricane extension

Specify the `ExposureMaps` block for the Hurricane extension

## Usage

``` r
insertExposureMaps(df)
```

## Arguments

- df:

  `data.frame` with columns `Degree`, `MapName`.

## Value

Character string(s) to write to the suitable LANDIS-II input file.
