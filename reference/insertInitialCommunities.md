# Specify `InitialCommunities` and `InitialCommunitiesMap` Files

Specify `InitialCommunities` and `InitialCommunitiesMap` Files

## Usage

``` r
insertInitialCommunities(files)
```

## Arguments

- files:

  character, specifying the paths to the initial communities and initial
  communities map files.

## Value

Character string(s) to write to the suitable LANDIS-II input file.
