# Specify a single `LandUse` block for the Land Use Plus extension

Specify a single `LandUse` block for the Land Use Plus extension

## Usage

``` r
insertLandUseType(lu)
```

## Arguments

- lu:

  `LandUseType` object.

## Value

Character string(s) to write to the suitable LANDIS-II input file.
