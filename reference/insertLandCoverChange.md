# Specify a single `LandCoverChange` block within a `LandUse`

Specify a single `LandCoverChange` block within a `LandUse`

## Usage

``` r
insertLandCoverChange(lcc)
```

## Arguments

- lcc:

  `LandCoverChange` object.

## Value

Character string(s) to write to the suitable LANDIS-II input file.
