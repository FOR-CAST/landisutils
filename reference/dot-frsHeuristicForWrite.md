# Translate a heuristic name to the upstream-parser form

Translate a heuristic name to the upstream-parser form

## Usage

``` r
.frsHeuristicForWrite(x)
```

## Arguments

- x:

  Character. A value from
  [.frsHeuristics](https://for-cast.github.io/landisutils/reference/dot-frsHeuristics.md).

## Value

Character. The lowercase-`"first"` form expected by the FRS extension's
input parser.
