# Specify Species Data File

Specify Species Data File

## Usage

``` r
insertSpeciesDataFile(file, core = NULL)
```

## Arguments

- file:

  Character, specifying the path to the file.

- core:

  logical, indicating whether to insert species input data file
  (`Species` if `TRUE`), or `SpeciesDataFile` (if `FALSE`) for use with
  succession extensions.

## Value

Character string(s) to write to the suitable LANDIS-II input file.
