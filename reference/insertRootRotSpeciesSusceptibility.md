# Specify the `SpeciesSusceptibility` block for the Root Rot extension

Specify the `SpeciesSusceptibility` block for the Root Rot extension

## Usage

``` r
insertRootRotSpeciesSusceptibility(df)
```

## Arguments

- df:

  `data.frame` with columns `Species`, `Index1`, `Index2`.

## Value

Character string(s) to write to the suitable LANDIS-II input file.
