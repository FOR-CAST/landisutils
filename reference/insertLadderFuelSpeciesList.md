# Specify `LadderFuelSpeciesList` for Social-Climate-Fire extension

Specify `LadderFuelSpeciesList` for Social-Climate-Fire extension

## Usage

``` r

```

## Arguments

  Character. Species names.

## Value

Character string(s) to write to the suitable LANDIS-II input file.
