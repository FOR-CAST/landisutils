# Recognized output-keyword names for the PnET Output extension

Names accepted by
[OutputBiomassPnET](https://for-cast.github.io/landisutils/reference/OutputBiomassPnET.md)'s
`Outputs` argument; mirrors the upstream parser in
`Extension-Output-Biomass-PnET/src/InputParametersParser.cs`.

## Usage

``` r
.pnetOutputKeywords
```

## Format

An object of class `character` of length 43.
