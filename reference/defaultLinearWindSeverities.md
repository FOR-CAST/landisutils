# Default `WindSeverities` table for the Linear Wind extension

Matches the example in the LANDIS-II Linear Wind Disturbance v3 User
Guide.

## Usage

``` r
defaultLinearWindSeverities()
```

## Value

`data.frame` with 4 columns: `Severity`, `LowerAge`, `UpperAge`,
`WindspeedMortalityThreshold`.
