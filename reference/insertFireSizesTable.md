# Specify Fire Size Table

Specify Fire Size Table

## Usage

``` r
insertFireSizesTable(df = NULL)
```

## Arguments

- df:

  data.frame corresponding to Fire Sizes Table

## Value

Character string(s) to write to the suitable LANDIS-II input file.
