# Prepare Biomass Succession Extension `FireReductionParameters` Table

Prepare Biomass Succession Extension `FireReductionParameters` Table

## Usage

``` r
prepFireReductionParameters(df = NULL)
```

## Arguments

- df:

  `data.frame` with columns `FireSeverity`, `WoodReduction`, and
  `LitterReduction`.

## Value

data.frame
