# Disturbance pool reductions used in `DisturbanceReductions`

Rows of the PnET-Succession `DisturbanceReductions` table (§10 of the
user guide). Each row specifies how a disturbance type partitions
biomass into / out of the dead pools.

## Usage

``` r
.pnetDisturbancePools
```

## Format

An object of class `character` of length 5.
