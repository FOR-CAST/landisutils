# Specify the `BDASpeciesParameters` table

Specify the `BDASpeciesParameters` table

## Usage

``` r
insertBDASpeciesParameters(df)
```

## Arguments

- df:

  `data.frame` with the columns enumerated in
  `.bdaSpeciesParameterCols`.

## Value

Character string(s) to write to the suitable LANDIS-II input file.
