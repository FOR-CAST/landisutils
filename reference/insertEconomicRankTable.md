# Specify the `EconomicRankTable` for a Biomass Harvest prescription

Specify the `EconomicRankTable` for a Biomass Harvest prescription

## Usage

``` r
insertEconomicRankTable(df)
```

## Arguments

- df:

  `data.frame` with columns `Species`, `EconomicRank`, `MinimumAge`.

## Value

Character string(s) to write to the suitable LANDIS-II input file.
