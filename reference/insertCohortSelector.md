# Specify a single cohort-selector line within a `LandCoverChange`

Specify a single cohort-selector line within a `LandCoverChange`

## Usage

``` r
insertCohortSelector(sel)
```

## Arguments

- sel:

  `CohortSelector` object.

## Value

Character string(s) to write to the suitable LANDIS-II input file.
